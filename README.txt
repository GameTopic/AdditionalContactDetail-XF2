Additional Contact Details (XF 2.3)

Install (first time)
1. Upload the contents of the /upload directory to your XenForo root.
2. In the Admin CP, go to Add-ons.
3. Click "Install/upgrade from archive" or "Install add-on".
4. Select "TylerAustin/AdditionalContactDetails" and confirm.

Upgrade
1. Upload the new files, overwriting existing ones.
2. In the Admin CP, go to Add-ons.
3. Click "Upgrade" for "Additional Contact Details".

Notes
1. After upgrade, user field metadata is re-applied (display order, validation callbacks, etc.).
